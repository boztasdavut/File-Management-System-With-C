# File-Management-System-With-C
C dili kullanılarak aynı linux'ta olduğu gibi bir dosya sistemi kodunu sizinle paylaşıyorum.
Linux'ta terminalde bulunan ls, mkdir, touch, nano, cat, rmdir, rm-r gibi komutları isimleri farklı olabilmekle beraber işlevsel olarak aynı şekilde çalıştırabilmektedir.
Dosya sistemi dallanmaya C:\\ dizini üzerinden başlamaktadır. Lütfen Programı kullanırken anlık komut çalışmalarını görebilmek ve izlemek için C:\\ dizin adresini açınız.

I am sharing a file system code with you using C language, just like in linux.
This code can run commands such as ls, mkdir, touch, nano, cat, rmdir, rm-r in the terminal in Linux, functionally the same, although their names may differ.
The file system starts branching from the C:\\ directory. Please open C:\\ directory address to see and watch instant command work while using the program.
